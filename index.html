--------:TYPES PSEUDO-CLASS SELECTOR:-------

Selects elements in a specific state

  • Structural Pseudo-classes

  1. :first-child: Selects the first child of a parent element.
  2. :last-child: Selects the last child of a parent element
  3. :nth-child(n): Selects the nth child of a parent element.
  4. :nth-of-type(n): Selects the nth child of a specific type within a parent.
  5. :first-of-type: Selects the first child of a specific type within a parent.
  6. :last-of-type: Selects the last child of a specific type within a parent.
  7. :only-of-type: Selects an element that is the only child of its type within its parent.
  8. :empty: Selects elements that have no children (including text nodes).

  • Link Pseudo-classes

  1. :link: Selects unvisited links
  2. :visited: Selects visited links.
  3. :hover: Selects elements when the user designates them (e.g., by placing a cursor over them).
  4. :active: Selects elements when they are being activated (e.g., during a mouse click).

  • Form Pseudo-classes

  1. :focus: Selects elements that have focus.
  2. :enabled: Selects enabled form elements.
  3. :disabled: Selects disabled form elements
  4. :checked: Selects checked elements (e.g., checkboxes or radio buttons)
  5. :indeterminate: Selects elements that are in an indeterminate state.
  6. :placeholder-shown: Selects input elements that are displaying placeholder text
  7. :valid: Selects elements with valid contents
  8. :invalid: Selects elements with invalid contents.
  9. :in-range: Selects elements with values within a specified range.
  10. :out-of-range: Selects elements with values outside a specified range

  • Other Pseudo-classes

  1. :not(selector): Selects elements that do not match the selector.
  2. :root: Selects the documentâ€™s root element
  3. :lang(language): Selects elements with a specific language attribute.
  4. :target: Selects the target element of a URL containing a fragment identifier
  5. :nth-last-child(n): Selects the nth child from the end of a parent element.
  6. :nth-last-of-type(n): Selects the nth child of a specific type from the end of a parent element.
  7. :dir(direction): Selects elements based on the directionality of the text.
  8. :fullscreen: Selects elements that are in fullscreen mode
  9. :read-only: Selects elements that are read-only
  10. :read-write: Selects elements that are editable.
