--------:TYPES PSEUDO-CLASS SELECTOR:-------

Selects elements in a specific state

  1. :hover: Applies when the user designates an element (e.g., by placing a cursor over it).
  2. :active: Applies while an element is being activated (e.g., during a mouse click).
  3. :focus: Applies while an element has focus (e.g., a form input).
  4. :visited: Applies once a link has been visited.
  5. :first-child: Applies to the first child of a parent element.
  6. :last-child: Applies to the last child of a parent element.
  7. :nth-child(n): Applies to the nth child of a parent element.
  8. :not(selector): Applies to elements that do not match the selector.
  9. :checked: Applies to elements that are checked (e.g., checkboxes or radio buttons).
  10. :disabled: Applies to elements that are disabled.
