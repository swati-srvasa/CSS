--------:TYPES PSEUDO-CLASS SELECTOR:-------

Selects elements in a specific state

  • Link Pseudo-classes

  1. :link: Selects unvisited links
  2. :visited: Selects visited links.
  3. :hover: Selects elements when the user designates them (e.g., by placing a cursor over them).
  4. :active: Selects elements when they are being activated (e.g., during a mouse click).
